## **属性**
- **`【target 表】`**

| 属性名          | 中文     | 最小长度 | 最大长度 | 类型           | 特殊要求
| --------------- | ------   | -------- | -------- | ---------      | --------
| **Tid**         | 标签id   | -        | -        | int            | -
| **Tfather**     | 父标签id  | -        | -        | int            | 每个标签（除了根root）要有一个父标签，父标签类型必须为 1
| **Tname**       | 标签名    | 1        | 30       | char           | -
| **Ttype**       | 标签类型  | 1        | 1        | int            | 1 为中间结点，2为叶子结点


---

## **接口 · 增加标签**
- **请求方法：POST**
- **接口网址：http://icpc-system.and-who.cn/Target/register**
- **表单要求**

| 属性名          | 必要性 | 最小长度 | 最大长度 | 特殊要求
| --------------- | ------ | -------- | -------- | --------
| **Utoken**      | O      | -        | -        | -
| **Tfather**     | X      | -        | -        | 无父结点则生成根结点
| **Tname**       | O      | 1        | 30       | -
| **Ttype**       | O      | 1        | 1        | -


- **示例**
```
{
	"Tfather": 1,
	"Tname": "字符串",
	"Ttype": 2
}
```

- **成功返回**
```
{
	"type": 1,
	"message": "增加成功",
	"data": []
}
```
